0) 게임 한 줄 요약

섰다는 화투패 중 1월부터 10월까지의 20장을 사용하여 2장을 투합한 족보(조합)로 승패를 가리는 베팅 게임입니다. "끗발이 선다"에서 유래되었으며, 심리전과 운, 그리고 대담함이 중요한 게임입니다.

1) 구성물 (패/숫자)
1-1. 사용 패 (총 20장)

화투의 1월부터 10월까지의 패, 총 20장만 사용합니다.
- 각 월(숫자)마다 2장씩 존재합니다. (예: 1월 광, 1월 홍단)
- 11월(똥)과 12월(비)은 사용하지 않습니다.

1-2. 패의 종류 (광/열/띠/피)
- 광(光): 1월 송학, 3월 벚꽃, 8월 공산명월 (총 3장)
- 열/띠/피: 나머지 17장
- 섰다에서는 주로 월(숫자)이 중요하며, '광'이 포함된 조합이 가장 강력한 족보를 만듭니다.

2) 족보 (강함 순위)
족보는 크게 38광땡 > 13/18광땡 > 땡 > 중간 족보 > 끗 > 망통 순서입니다.

2-1. 최강 족보 (천하무적)
- 38광땡: 3월 광 + 8월 광. 섰다에서 가장 강력한 족보입니다. 그 어떤 족보도 이길 수 없습니다.
- 13광땡/18광땡: 1월 광 + 3월 광 / 1월 광 + 8월 광. 38광땡 다음으로 강력하며, 서로 간의 우열은 없습니다(동급).

2-2. 땡 (같은 월 2장)
같은 월의 패 2장을 모으면 '땡'이라고 하며, 숫자가 높을수록 강력합니다.
- 장땡 (10땡): 10월 + 10월 (최고)
- 9땡 ~ 1땡: 숫자가 클수록 이깁니다. (9땡 > 8땡 > ... > 1땡(삥땡))

2-3. 중간 족보 (특수 조합)
일반 끗수 계산보다 우선하는 특수 조합들입니다.
- 알리 (1월 + 2월): 땡 잡는다고 불리지만 실제로는 땡 아래 최강 조합.
- 독사 (1월 + 4월): 알리 다음 순위.
- 구삥 (9월 + 1월): 독사 다음.
- 장삥 (10월 + 1월): 구삥 다음.
- 장사 (10월 + 4월): 장삥 다음.
- 세륙 (4월 + 6월): 장사 다음.

2-4. 끗 (일반 계산)
위의 족보에 해당하지 않는 경우, 두 패의 월 숫자를 더한 값의 **일의 자리 수**가 '끗'이 됩니다.
- 갑오 (9끗): 합의 일의 자리가 9 (가장 높음)
- 8끗 ~ 1끗: 숫자가 높을수록 유리
- 망통 (0끗): 합의 일의 자리가 0 (가장 낮음, 꽝)
- 예시: 3월 + 8월(광 아님) = 11 -> 1끗 / 1월 + 8월(광 아님) = 9 -> 갑오

3) 특수 규칙 (변수 창출)
게임의 재미를 위해 특정 조건에서 발동하는 특수 규칙들이 있습니다.

3-1. 암행어사 (4월 열/띠 + 7월 열/띠)
- 4월 열/띠와 7월 열/띠의 조합입니다.
- 13광땡, 18광땡을 잡을 수 있습니다. (암행어사 승)
- 단, 상대방에 광땡이 없으면 단순 '1끗'으로 취급됩니다.
- 38광땡은 잡을 수 없습니다.

3-2. 땡잡이 (3월 광 + 7월 열/띠)
- 3월 광과 7월 열/띠의 조합입니다.
- 1땡(삥땡)부터 9땡까지 모든 '땡'을 잡습니다. (장땡 및 광땡은 못 잡음)
- 상대방에 땡이 없으면 '망통(0끗)'으로 취급됩니다.

3-3. 구사 (9월 + 4월) / 멍텅구리 구사 (9월/4월 열/띠 조합)
- 최고 족보가 '알리' 이하일 때 (땡 이상이 없을 때), 이번 판을 무효화하고 재경기를 요청할 수 있습니다 (판돈 유지).
- 멍텅구리 구사(열/띠로만 이루어진 9+4)의 경우, 9땡 이하까지 재경기를 만들 수 있습니다 (규칙에 따라 다름).

4) 게임 진행 방식 (베팅)

4-1. 기본 흐름
- 참가자들은 기본 판돈(앤티)을 냅니다.
- 각자 패를 1장씩 받습니다 (보여주거나 숨김). (생략하고 2장 다 받고 시작하기도 함)
- 베팅 진행 (1차)
- 나머지 1장을 받습니다.
- 베팅 진행 (2차 - 메인 베팅)
- 족보 공개 및 승패 결정

4-2. 베팅 용어
- 다이(Die): 이번 판 포기.
- 콜(Call): 앞사람이 건 금액만큼만 걸고 진행.
- 체크(Check): 돈을 걸지 않고 순서를 넘김 (선만 가능하거나, 아무도 베팅 안 했을 때).
- 삥(Ping): 기본 단위만큼만 베팅 (주로 선이 탐색전으로).
- 따당(Ddadang): 앞사람인 건 금액의 2배를 베팅.
- 하프(Half): 전체 판돈의 절반을 베팅.
- 풀(Full/All-in): 베팅 한도 끝까지 혹은 가진 전재산.

5) 온라인 구현 팁

5-1. 족보 판정 로직
- 두 카드의 월(month)과 광(isKwang) 여부를 속성으로 가집니다.
- if (is3Kwang && is8Kwang) return '38GwangDdang';
- else if (is1Kwang && (is3Kwang || is8Kwang)) return 'GwangDdang';
- else if (month1 == month2) return month1 + 'Ddang';
- ... (중간 족보 체크)
- else return (month1 + month2) % 10 + 'Ddot';

5-2. AI 구현
- 족보의 승률 데이터(Expected Value)를 기반으로 베팅 성향(공격적/방어적)을 설정합니다.
- 예: 38광땡은 무조건 하프/풀, 망통은 다이/뻥카(Bluffing) 시도.

5-3. 연출
- 화투패를 쪼는 맛(Peeking)을 마우스 드래그 등으로 구현하면 몰입감이 극대화됩니다.
- '타짜' 영화의 대사나 분위기를 차용하면 재미요소가 됩니다. ("싸늘하다..", "묻고 더블로 가!")
